
# ВТББлаго

**ВТББлаго** — это платформа для помощи нуждающимся.  
Создавай проекты, поддерживай инициативы и отслеживай результаты благотворительности.

---

## Как запустить проект

### 1. Требования
- Python 3.10+  

### 2. Установка зависимостей
Сначала установите все библиотеки:
```bash
pip install -r requirements.txt
````

### 3. Запуск

Для запуска сервера разработки:

```bash
python manage.py runserver
```

Откройте в браузере [http://127.0.0.1:8000](http://127.0.0.1:8000)

### 4. Настройки

* Основные настройки — в `settings.py`
* Можно использовать переменные окружения для секретных данных:

  * `SECRET_KEY` — секретный ключ проекта
  * `DEBUG` — включить/выключить режим отладки
  * `DATABASE_URL` — путь к вашей базе данных

### 5. Учётные данные

Чтобы зайти в админику, создайте суперпользователя:

```bash
python manage.py createsuperuser
```
и откройте
[http://127.0.0.1:8000/admin](http://127.0.0.1:8000/admin/)
### 6. Структура проекта

* `manage.py` — запуск проекта
* `app/` — основное приложение с логикой и моделями
* `templates/` — HTML-шаблоны
* `static/` — CSS, JS, картинки
